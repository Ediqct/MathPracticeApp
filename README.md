Math Practice Application
A Python program that generates randomized arithmetic problems across multiple difficulty levels. Includes addition, subtraction, multiplication, and division with logic to ensure valid results. Designed to help users practice math skills while demonstrating clean program structure and modular function design.

Features
- Difficulty selection (easy, medium, hard)
- Randomized problem generation
- Clean modular functions
- Valid number generation for subtraction and division
- Answer validation and feedback

How to Run
- Install Python 3.10 or later
- Download or clone the repository:
git clone https://github.com/Edicqt/MathPracticeApp
- Navigate into the project folder
- Run the program:
python main.py
